# Changelog

All notable changes to this project are documented in this file.

## v1.0.0

- Public author website with editorial design language.
- Route architecture for home, about, library, blog, reviews, contact, search, legal pages, and 404.
- Admin CMS with protected routes for dashboard, books, blog, reviews, messages, newsletter, media, and settings.
- Supabase integration for auth, content, reviews, messages, newsletter, settings, and media storage.
- Runtime fallback system for local data and local storage when Supabase or external endpoints are unavailable.
- Buy lead capture flow with retailer tracking and optional challenge token support.
- Review moderation pipeline with submitted and published states.
- Contact and newsletter flows with multi-channel fallback behavior.
- SEO system including metadata, structured data, sitemap, and robots generation.
- Production security hardening including CSP and standard security headers.
- Centralized URL safety sanitization for external and dynamic links.
- Accessibility hardening for active dialogs and keyboard navigation.
- Mobile responsive layouts and optimized media pipeline.
